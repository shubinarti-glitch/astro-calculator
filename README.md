# Астрокалькулятор — натальная карта и транзиты

Веб-сайт для расчёта натальных карт и транзитов (аналог geocult.ru).
Расчёты выполняет **Swiss Ephemeris** через библиотеку **Kerykeion** —
профессиональная точность положений планет, домов и аспектов.

## Возможности

- 🪐 **Натальная карта**: положения планет, угловые точки (ASC/MC/DSC/IC),
  12 домов, аспекты, фаза Луны, SVG-колесо карты.
- 🔭 **Транзиты**: положение планет на любую дату и их аспекты к натальной карте,
  с двойным колесом (натал + транзит).
- 💞 **Синастрия**: совместимость двух карт — межкартные аспекты, двойное колесо
  и индекс совместимости (метод Ч. Дисчеполо).
- ☀️ **Соляры и лунары**: карты солнечного и лунного возвращения на заданный год.
- ⏳ **Прогрессии и дирекции**: вторичные прогрессии (день=год) + дирекции
  солнечной дуги с аспектами к наталу.
- 🧭 **Портрет личности**: профессиональный синтез Солнца, Луны и Асцендента,
  управитель карты, психологический профиль по балансу стихий и крестов.
- ⚖️ **Эссенциальные достоинства** планет (обитель, экзальтация, изгнание, падение).
- 📜 **Текстовые интерпретации**: развёрнутые трактовки планет в знаках и домах,
  характер аспектов — на уровне профессионального разбора.
- 📅 **Эфемеридный календарь транзитов**: даты точных транзитов медленных планет
  к натальной карте за период (до 2 лет).
- 🔮 **Прогноз на период**: годовая профекция (хозяин года), прогрессивная Луна
  и ключевые транзиты медленных планет с трактовкой тем и сроков. Прогноз описывает
  астрологические тенденции, а не предопределённые события.
- 🕰️ **Ректификация времени рождения**: подбор неизвестного времени по событиям
  жизни — перебор кандидатных времён с оценкой попадания солнечно-дуговых дирекций
  и транзитов на углы карты (ASC/MC), рейтинг вероятных времён.
- 👤 **Личный кабинет**: регистрация/вход, сохранение и загрузка карт (SQLite).
- 🌍 **Поиск города** через OpenStreetMap + офлайн-определение часового пояса
  по координатам (timezonefinder).
- 🏠 8 систем домов (Плацидус, Кох, Цельнознаковая, Региомонтан и др.).
- Полностью русский интерфейс.

## Установка

```bash
python -m venv venv
venv\Scripts\activate          # Windows
pip install -r requirements.txt
```

## Запуск

```bash
python run.py
```

Откройте в браузере: <http://127.0.0.1:8000>

## Архитектура

```
backend/
  ephe.py        — настройка пути к эфемеридам (обход проблемы не-ASCII путей на Windows)
  constants.py   — русские названия знаков, планет, домов, аспектов
  astrology.py   — ядро расчётов (натал, транзиты) на Kerykeion
  main.py        — FastAPI: API + раздача фронтенда
frontend/
  index.html     — интерфейс
  css/style.css  — оформление (космическая тёмная тема)
  js/app.js      — логика: форма, автодополнение, рендер таблиц и карты
```

## API

| Метод  | Эндпоинт              | Назначение                                  |
|--------|-----------------------|---------------------------------------------|
| POST   | `/api/natal`          | Натальная карта                             |
| POST   | `/api/transit`        | Транзиты на дату + аспекты к наталу         |
| POST   | `/api/synastry`       | Синастрия двух карт + индекс совместимости  |
| POST   | `/api/return`         | Соляр / лунар (карта возвращения)           |
| POST   | `/api/progression`    | Прогрессии + дирекции солнечной дуги        |
| POST   | `/api/calendar`       | Календарь точных транзитов за период        |
| POST   | `/api/forecast`       | Прогноз: профекция + прогр. Луна + транзиты  |
| POST   | `/api/rectification`  | Подбор времени рождения по событиям жизни    |
| POST   | `/api/auth/register`  | Регистрация                                 |
| POST   | `/api/auth/login`     | Вход                                        |
| GET    | `/api/auth/me`        | Текущий пользователь                        |
| GET    | `/api/profiles`       | Список сохранённых карт                     |
| POST   | `/api/profiles`       | Сохранить карту                             |
| DELETE | `/api/profiles/{id}`  | Удалить карту                               |
| GET    | `/api/geocode`        | Поиск города (Nominatim) + часовой пояс     |
| GET    | `/api/meta`           | Справочные данные (системы домов)           |

## Примечание про эфемериды

На Windows Swiss Ephemeris не открывает файлы `.se1` по путям с кириллицей.
Модуль `backend/ephe.py` автоматически копирует файлы эфемерид в ASCII-папку
(`~/.astro_ephe` или `C:\sweph`) и перенаправляет туда библиотеку — отдельных
действий от пользователя не требуется.
